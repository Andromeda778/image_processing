Есть несколько методов выделения контуров и границ в изображениях 
Все методы основаны на свойстве сигнала яркости - разрывности.
Наиболее общий метод поиска разрывов - пространственная фильтрация

1.Метод Робертса - самый простой алгоритм, который вычисляет на плоском дискретном 
    изображении сумму квадратов разниц между смежными пикселями. Использует ядра 2х2

2.Метод Превитт вычисляет максимальный отклик на множестве ядер свёртки
        для нахождения локальной ориентации границы в каждом пикселе. Два ядра 3 х 3

3.Метод Собеля - оператор, вычисляющий приближённое значение градиента
   яркости изображения. Отличие от Превитта заключается в использовании весового коэффициента 2 для средних
элементов. Использует ядра 3 х 3

4. Метод Шарра является модифицированной версией оператора Собеля, тоже использует ядра 3х3

5. Детектор границ Канни представляет собой более комплексный алгоритм, включающий в
   себя предварительную обработку изображения для уменьшения шумов. Применяет ядра 5х5


Свёртка изображения - это операция вычисления нового значения 
выбранного пикселя, учитывающая значения окружающих его пикселей.
При вычислении нового значения выбранного пикселя изображения, ядро свёртки прикладывается своим центром к этому пикселю.
Соседние пиксели так же накрываются ядром.
Далее, вычисляется сумма произведений значений пикселей изображения на значения, накрывшего данный пиксель элемента ядра.
Полученная сумма и является новым значением выбранного пикселя.



3. Ядра фильтра представляют собой значения яркости некоторого элемента изображения 
 Используется для применения свёртки. При увеличении размера ядра 
фильтра итоговое изображение фильтра будет более размытым, и соответственно, наоборот.
